# 标题字体：思源宋体

请将思源宋体（Source Han Serif）字体文件放入此目录：

- **SourceHanSerifCN-Regular.otf** — 正文
- **SourceHanSerifCN-Bold.otf** — 高亮加粗

下载地址（Adobe 官方）：  
https://github.com/adobe-fonts/source-han-serif/releases  

选择 **Source Han Serif CN**（简体中文）子集或完整 OTF 包，解压后把上述两个文件复制到本目录即可。

若未放置字体文件，将回退为系统 sans-serif。
